### 🌟 Featured Projects

#### 🚀 **[RepoInsight](https://github.com/shrish98/RepoInsight)**
An intelligent, full-stack application that lets users clone, index, and **chat directly with GitHub repositories** using semantic vector search and cognitive AI agents.
* **Core Features:** Automatically chunks codebases, generates Gemini AI embeddings, indexes into MongoDB Atlas Vector Search, and orchestrates answers via a LangGraph agent.
* **Stack:** React 19, Vite, Tailwind CSS v4, Node.js, Express, MongoDB, Docker, AWS EC2.
* 👉 **Live Demo:** [http://13.201.51.249:5000](http://13.201.51.249:5000)

#### 📉 **[DropAlert](https://github.com/shrish98/dropalert)**
A price-tracking SaaS that monitors products on any e-commerce site and automatically alerts users when prices drop.
* **Core Features:** Scrapes store sites with Firecrawl, stores dynamic price data in Supabase, schedules automated daily price-checks via Upstash Redis/Cron, and alerts users via Resend.
* **Stack:** Next.js 16, Supabase, Redis, Resend, Firecrawl, Recharts, Tailwind CSS v4.
* 👉 **Live Demo:** [https://dropalert-lemon.vercel.app/](https://dropalert-lemon.vercel.app/)

#### 🎓 **[PrepWise (AI Interview Prep)](https://github.com/shrish98/AI-Interview-Prep)**
An AI-driven career prep SaaS offering interactive mock interviews, behavioral quizzes, and automatic resume builders.
* **Core Features:** Uses Google Gemini API to conduct conversational mock interviews, generates custom dashboard metrics, schedules tasks via Inngest background queues, and exports resumes/feedback logs to PDF.
* **Stack:** Next.js 16, Prisma ORM, Clerk Auth, Inngest, Google Gemini API, Recharts, Tailwind CSS v4.
* 👉 **Live Demo:** [https://ai-interview-prep-rose.vercel.app/](https://ai-interview-prep-rose.vercel.app/)
